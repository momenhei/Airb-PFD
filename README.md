# Airbus-PFD
Dieses Projekt immitiert das [Primary Flight Dispaly (PFD)](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/pfd/) der Airbus A320 Familie.
Dabei ist das Display für den Einsatz im Auto konzipiert und übertragt für den Straßenverkehr relevante Daten in das typische Cockpit-Layouts.
## Vorraussetzungen
Zur Nutzung wird die SDL3 Bibliothek benötigt. Die Dokumentation sowie eine Anleitung für die Installation sind [hier](https://wiki.libsdl.org/SDL3/FrontPage) zu finden.
